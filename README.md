GENERADOR SEGURO DE CONTRASEÑAS
OBJETIVO DEL PROGRAMA:
  Crear contraseñas fuertes, aleatorias y difíciles de adivinar o descifrar, con el fin de proteger cuentas, sistemas y datos frente a accesos no autorizados.
  
INTRODUCCION AL PROYECTO:
  Para ser capaz de tener una contraseña robusta y aleatoria capaz de protejer el acceso a las cuentas, se crear el software de esta aplicacion. 
  En la que se utiliza python v.3.13 junto con las librerias de:
    * RANDOM (seleccion aleatoria de caracteres)
    * HASHLIB (cifrado de la contraseña) 
    * TKINTER (para la visualizacion de la aplicacion en una GUI)
FECHA:
  21/12/2025
  
PRINCIPALES FUNCIONALIDADES DEL CODIGO:

  Primero se le pregunta el nombre al cliente, junto con que caracteristicas desea que se incluyan en su contraseña, tales como mayusculas, minusculas, simbolos, etc. Lugo si desea generar la contraseña, selecciona el boton de generar contraseña. Y por ultimo se muestra su contraseña generada junto con su version cifrada en caso de que haya seleccionado que si. (La contraseña cifrada es mas que nada para que se añada a una base de datos si el cliente lo desea)
  - Uso de bucles, librerias, tuplas y funciones
  - RANDOM (seleccion aleatoria de caracteres)
  - HASHLIB (cifrado de la contraseña) 
  - TKINTER (para la visualizacion de la aplicacion en una GUI)
