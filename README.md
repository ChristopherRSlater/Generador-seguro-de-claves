# Este proyecto es un software de un generador seguro de contraseña en Python v3.13
# El programa inica con una bienvenida (con nombre del cliente), y una pregunta de si desea generar una contraseña, le pregunta que caracteristicas desea en su contraseña y al final le muestra cual fue la contraseña generada
#Dentro de la logica del programa esta implementado la biblioteca random para asegurar la aleatoriedad de los caracteres de la contraseña
